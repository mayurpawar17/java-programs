1. Payment Gateway System (Easy)
   Requirements

An e-commerce application supports multiple payment methods.

Current payment methods:

Credit Card
UPI
PayPal

Future payment methods:

Apple Pay
Google Pay
Crypto

Some payment methods support refunds.

Some support recurring payments.

The system should also send notifications after successful payment.

Notifications can be:

Email
SMS
WhatsApp
Your Task

Design classes/interfaces.

Think about:

Which interfaces are needed?
Which classes should implement them?
How will you add new payment methods?
How will Notification work?
Which classes depend on abstractions?
